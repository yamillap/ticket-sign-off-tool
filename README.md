# Ticket Sign Off Tool

## Overview

A full-stack ticket automation tool that generates formatted QA/testing sign-off tickets through a modern React frontend and Spring Boot backend API.

## Features

- Generate formatted sign-off tickets
- Copy generated tickets to clipboard
- Responsive glassmorphism-inspired UI
- React state management with hooks
- Spring Boot REST API integration
- JSON request handling between frontend and backend


## Current Functionality

- Fillable form of ticket information through the UI
- Submit ticket data to the backend API
- Generate structured sign-off outputs
- Copy generated tickets instantly
